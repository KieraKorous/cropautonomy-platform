# CropAutonomy / GaiaBots Documentation Index

This folder is the operating context for AI agents, engineers, designers, and future collaborators building the CropAutonomy ecosystem.

The first implementation goal is to launch focused landing pages for:

- `cropautonomy.com`
- `gaiabots.ai`

Those landing pages are only the starting point. The documentation in this folder should guide the project toward a working multi-tenant prototype targeted for August 2026 and, beyond that, a full autonomous agricultural intelligence platform.

## Documentation Principles

- Keep docs future-facing, not limited to the current landing pages.
- Treat CropAutonomy as the primary platform and GaiaBots as the robotics/device brand.
- Assume multi-tenant organizations from the beginning.
- Build the web foundation so it can later support devices, telemetry, AI analysis, field mapping, and operations.
- Avoid generic SaaS patterns when they conflict with field reliability, robotics integration, or agricultural workflows.
